---------------------------------------------------
! Tout ce qui va suivre est a exécuter sous Linux ! 
---------------------------------------------------

Vous avez un fichier '.evaluation.c', c'est ce programme qui permet de réaliser l'évaluation des performances de l'IA. 

Dans le dossier 'code', vous trouverez le fichier 'MaStrategie.c', correspondant à mon IA.
Vous trouverez aussi 2 autres fichiers 'Strategie_Test1.c' et 'Strategie_Test2.c', qui sont 2 autres IAs, non développées par moi-même.

Pour réaliser l'évaluation, il vous suffit de compiler ces 3 fichiers. 
Une fois fait, vous devez placer les exécutifs dans le dossier 'strats'. 
Par la suite, vous devez compiler le fichier '.evaluation.c', puis l'exécuter. 
Enfin, il ne vous reste plus qu'a suivre les instructions dans la console.
